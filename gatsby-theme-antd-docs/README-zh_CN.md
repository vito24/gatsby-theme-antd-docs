# gatsby-theme-antd-docs

[English](./README.md) | 简体中文

## 快速开始

```sh
yarn add gatsby-theme-antd-docs
```

添加主题到你的 `gatsby-config.js`. 

```javascript
module.exports = {
  plugins: ['gatsby-theme-antd-docs']
}
```

就这么简单，现在你可以执行下面的命令来运行你的网站

```sh
yarn gatsby develop
```

打开浏览器访问 http://127.0.0.1:8000.
